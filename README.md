# 🌌 VoidRepo: Bulk Manager

[Gemini Link](https://gemini.google.com/app/82a079787eca3ce7)

A high-performance, dark-aesthetic GitHub repository management tool built with **Vite+** and **Next.js**. VoidRepo allows power users to filter, select, and modify hundreds of repositories in seconds.

## 🚀 Key Features
- **Bulk Actions:** Delete, Archive, or Change Visibility (Public/Private) for multiple repos at once.
- **Advanced Filtering:** Filter by stale status, size, language, or license.
- **Safety First:** Multi-step confirmation and "Type-to-Confirm" protection for destructive actions.
- **VoidZero Powered:** Ultra-fast builds and HMR using the `vp` binary.

## 🛠 Tech Stack
- **Framework:** Next.js 15+ (App Router)
- **Tooling:** [Vite+](https://viteplus.dev/) (vp)
- **Auth:** Auth.js (GitHub OAuth)
- **SDK:** Octokit.js
- **State:** TanStack Query & Zustand
- **UI:** Tailwind CSS + Shadcn UI (Dark Theme)
